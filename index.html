Project Idea

Overview :

 The product is about a smart mirror displayed in the clothing store which contains a variety of functions. First, after customers decides what clothes, dresses or pants they want, rather than going into fitting rooms, this time, customers only need to take what they have decided to the mirror. Then the mirror will automatically show what do customers look like when they dress up. There are two basic ways to tell the smart mirror what you want it to display. Firstly, each goods contain a special code, like barcode, so all customers have to do is to scan the code. Secondly, customers can search products on smart mirror and find what they want. Every time I go to clothing store, I spend more time in the fitting room rather than choosing cloth. But if there is a machine that can displays clothes on my body, it makes me have more time to browsing. If customers want to try a lot of clothes, they have to put it on and take it off again and again. Basically, trying clothes on is a time-consuming thing but if there is a mirror can display on customers’ body, it saves time for them. Smart mirror can save time and let customers having more time on chasing clothes.

Motivation :

 Every time when I went to a clothing store, I spent more time in the fitting room trying clothes on than shopping or deciding what I want in the store. However, if there is a machine that can display how I look like when I dress them up, it lets me have more time to browsing products. If customers want to try a lot of clothing, they have to dress it up and take it off again and again. Basically, trying clothes on is a time-consuming thing but if there is a mirror which can display how customers look like when they wear the clothing they just picked, it saves time for them. Smart mirror can save time and let customers have more time on choosing clothing.

Description :

 A smart mirror is a product that combine mirror and a touchable screen, and this is how it works. To begin with, a smart mirror contains database that store manager can enter goods into it. For example, enter the product names or number each time whenever there is new arrival. In addition, when searching products on smart mirror, products can be searched by names, product numbers or categories, such as clothing, accessories, kicks. It also contains recommendation for customers based on products they have browsed. After customers pick the clothing they want in the shop, walk to smart mirror, and then scan the barcode on tag or customers can find it according to category. Customers need to make sure the whole body is in smart mirror when scanning customers’ body shape for accurately displaying results on the screen. A benefit of having the smart mirror in the clothing store is if there are any goods that have sold out, customers still can find the sold-out products on the smart mirror, and have it display the appearance when they dress them up. This function will not make any customers down because of some hot selling sold out. Furthermore, customer can add what they want to buy to cart and then do the payment on the machine connected to the smart mirror. This function allows customers to save time on the steps of picking things, finding matched size and taking desired products to the cashier and checking out in the store. All these steps can be done in one machine, one place within short amount of time. Moreover, smart mirror can also recommend customers clothing size according to their body shapes and heights. A smart mirror can reduce a lot of time for customers. Overall, smart mirror is a wonderful product that can benefit both sides for customers and shops.

Tools and technologies :

 In order to make a smart mirror, firstly it needs a touchable screen. It also needs a scanner to scan code. One of the most important software is database. In order to successful scan products by customers, data of goods need to be stored into database to make scanning successfully happen. Another important thing is mirror itself. Payment system and equipment are also required for a smart mirror.

Skills required :

 In order to achieve requirements above, I need to be extremely familiar with database system such as SQLite Studio and Oracle 18c, because Oracle 18c is an amazing system for users and it allows many users to access at the same time.

Outcome :

 Once a smart mirror is successful made, it saves a lot of time for customers and employees. This is because, auto-payment reduces a lot of time for customers, and it reduces a ton of works for employees. It increases the numbers of customers, and it also makes more profits for stores. However, some employees have a high possibility to be replaced by the machine which contains auto-payment function and recommendation of related items function. The smart mirror has good sides and downsides for human beings.
